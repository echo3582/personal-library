Design a minimalist personal web app for managing a private paper book collection.

Style:
- Clean, calm, personal study room feeling
- Light background (warm off-white or very light gray)
- Elegant typography
- Plenty of white space
- Subtle borders, no heavy shadows
- Desktop-first responsive layout

App name: My Library

Core concept:
This is a personal library manager using Chinese Library Classification (CLC) main categories.
Focus on simplicity, clarity, and structure.
No location management.
No reading progress tracking.

Pages to design:

1) Dashboard (Home)
- Top navigation bar with:
  - App name on the left
  - Global search bar in the center
  - Add Book button on the right
- Main content:
  - Grid of CLC main category cards (e.g., I Literature, B Philosophy, K History, TP Computer Technology)
  - Each card shows:
    - Category code (large typography, e.g., "I")
    - Category name (e.g., Literature)
    - Number of books
  - Clean card layout, soft border, hover interaction
- Optional small section below:
  - Recently added books (simple horizontal list)

2) Library Page
- Left sidebar:
  - List of CLC main categories
  - Active state highlight
- Main area:
  - Filter bar:
    - Status filter chips (Unread / Reading / Read / Paused / Want to Read)
    - Tag filter
  - Book list as clean vertical cards:
    - Title (primary)
    - Author (secondary text)
    - Category badge (e.g., I, TP)
    - Status badge
    - Tags as subtle chips
  - Each card clickable

3) Add Book Page (Form)
- Centered narrow layout
- Fields:
  - Book Title (required)
  - Author
  - CLC Category (select with top common categories as quick buttons)
  - Status (default: Unread)
  - Tags (input with chip style)
  - Note (small textarea)
- Buttons:
  - Save
  - Save & Add Another

Design language:
- Serif or elegant sans-serif font pairing
- Neutral palette (cream, soft gray, muted green or deep blue accent)
- Rounded corners subtle (8px)
- No heavy gradients
- Calm and intellectual atmosphere

Output:
High-fidelity wireframes for desktop layout (1440px width).
